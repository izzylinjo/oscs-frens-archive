# OSCS & Frens Archive — Project Status
Last updated: session 1 (setup in progress)

## What this is
Automated YouTube Shorts bot that pulls top Twitch clips from OSCS and their
friend group, runs them through a Discord review step, then posts to YouTube
Shorts on a staggered schedule. Weekly and monthly compilations go to the main
channel as long-form videos.

## Tech stack
- Language: Python
- Clip source: Twitch public API (no streamer permission needed)
- Transcription: Whisper (local, free, runs on CPU)
- Title generation: Gemini free tier API
- Overlay: ffmpeg (streamer name in corner, small)
- Review: Discord bot DM (approve/reject/edit title)
- Database: SQLite (local archive of all approved clips)
- Upload: YouTube Data API v3
- Scheduler: APScheduler (30 min stagger between posts)
- Runtime now: Windows + VS Code (local dev)
- Runtime later: Oracle Free Tier VPS (24/7 production)

## Pipeline summary
1. Twitch API polls top clips 2-3 hrs after stream ends
2. Whisper transcribes each clip
3. Gemini generates suggested title from transcript
4. ffmpeg burns small streamer name overlay in bottom-left corner
5. Discord bot DMs you: preview link + suggested title + streamer credit
6. You react: check = approve, x = reject, reply = custom title
7. You can also manually submit a clip URL to the Discord bot anytime
8. Approved clips go into SQLite queue
9. APScheduler posts one clip every 30 mins to YouTube Shorts
10. Each clip description includes streamer Twitch, TikTok, IG, Twitter
11. Every approved clip is archived in SQLite regardless of when posted
12. End of week: ffmpeg compiles that week's clips → you approve → posts as long-form
13. End of month: same for monthly recap (e.g. "Top OSCS Clips of January")

## Key decisions made
- No vertical crop (9:16) for now — post landscape, revisit month 2
- No AI video trimming for now — Twitch clips are pre-trimmed
- No sound effect edits (not the style we want)
- Streamer credit: small corner overlay + full socials in description
- Streamer name in title only if it makes the title better (not forced)
- Rejected clips: archived in db, never posted
- YouTube Shorts only at launch — TikTok/Instagram/Twitter later
- Weekly + monthly compilations = long-form on main channel, not Shorts
- Future: themed compilations from archive (e.g. "every time a2guap goes insane")
- Friends list kept short for now — easy to expand in config.py
